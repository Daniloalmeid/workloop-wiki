# 🌐 Prospecção via Browser (Playwright)

> Método gratuito de prospecção usando o navegador diretamente no Google Maps.

## Como funciona

1. Hermes abre o Google Maps no navegador
2. Digita a busca (ex: "material de construção Jaraguá")
3. Extrai os resultados da lista lateral
4. Clica em cada resultado para ver detalhes (telefone, site)
5. Tenta encontrar Instagram via curl no site
6. Compila tabela e entrega no Telegram

## Vantagens

- ✅ **Grátis** — sem custo de API
- ✅ **Dados reais** — vê exatamente o que o Google Maps mostra
- ✅ **Flexível** — pode explorar organicamente
- ✅ **Sem setup** — não precisa instalar nada

## Desvantagens

- ❌ **Manual** — precisa de comando cada vez (sem cron)
- ❌ **Devagar** — ~2-3 min por busca
- ❌ **Limitado** — Google Maps sem login mostra ~20 resultados

## Quando usar

- Prospecção diária (1 nicho/dia)
- Quando o cliente pede algo muito específico
- Para explorar uma região nova

## Exemplo de comando

"Busca material de construcao em Jaragua Sao Paulo no Google Maps"
