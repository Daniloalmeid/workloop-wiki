# 🔍 Prospecção de Clientes

## Como encontramos clientes

| Método | Ferramenta | Status |
|---|---|---|
| Google Maps | Apify / Manual | ✅ Funcionando |
| Indicação | Rede de contatos | 📝 A fazer |

## Passo a passo

1. Buscar clínicas de estética no Google Maps
2. Coletar: nome, telefone, endereço, avaliações
3. Qualificar por porte/localização
4. Abordar via WhatsApp com proposta
5. Se interesse → [[Processos/Onboarding]]
