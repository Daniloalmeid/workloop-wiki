# 🆕 Onboarding de Cliente

## Checklist

- [ ] Criar cliente no Supabase (tabela `clientes`)
- [ ] Configurar perfil no Evolution API
- [ ] Coletar dados iniciais (serviços, preços, horários)
- [ ] Importar dados se houver planilha
- [ ] Testar fluxo de atendimento
- [ ] Testar agendamento
- [ ] Apresentar ao cliente

## Duração estimada
- **Setup técnico:** 1-2 horas
- **Importação de dados:** 1-2 horas
- **Validação com cliente:** 1 hora
